# homebridge-commander
Simple CLI plug-in for homebridge

# Supported type
- Switch
- Lightbulb

## Config.json settings


# Optional for Lightbulb
Set the listed settings to true to have them active in Homekit
'''
"brightness" : true,
"hue" : true,
"saturatiuon" : true,
"colortemperature" : true,
'''

# Optional for Speaker
Set the listed settings to true to have them active in Homekit
'''
"volume" : true,
'''
